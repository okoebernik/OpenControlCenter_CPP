\# OpenControlCenter Roadmap



A roadmap is not a list of tasks.

It is the vision of what can be built.



\## Version 0.1 - The Foundation



Goal: create a clean and stable software foundation.



Status: in progress



\- Application core

\- ScreenManager

\- EventBus

\- ServiceManager

\- LayoutManager

\- Theme system

\- Widget base

\- Dashboard screen

\- Settings screen



\## Version 0.2 - Alive



Goal: make the dashboard react to live system data.



Planned:



\- DashboardViewModel

\- live heap display

\- live PSRAM display

\- live flash information

\- live clock

\- status bar updates



\## Version 0.3 - Connected



Goal: connect OCC to the network.



Planned:



\- WiFiService

\- MQTTService

\- MQTT status

\- MQTT data model

\- first real MQTT card



\## Version 0.4 - Insight



Goal: visualize data history.



Planned:



\- charts

\- gauges

\- history graphs

\- trend indicators



\## Version 0.5 - Modules



Goal: add real application modules.



Planned:



\- Aquarium module

\- Victron module

\- Weather module

\- Synology module

\- Proxmox module

\- Fire department module



\## Version 1.0 - Control Center



Goal: stable, usable and extensible embedded control platform.
